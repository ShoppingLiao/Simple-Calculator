# Simple-Calculator

簡單的計算機

注意事項：

- 需有 RWD：
  - 裝置解析度大於等於 1024 Pixel 時，頁面設計之寬度解析度以 1024 Pixel 呈現。
  - 裝置解析度小於 1024 Pixel 時，頁面設計之寬度解析度以 360 Pixel 呈現。
- PC 支援瀏覽器：
  IE11，Chrome、Firefox、Safari 以最新版本進行測試
- 行動裝置支援的作業系統、瀏覽器版本：
  - 作業系統 iOS9~10 內建瀏覽器 Safari
  - 作業系統 Android5.0~6.0 的瀏覽器 Chrome
- 圈圈的數字要在正中間
- 僅需有加、減的運算功能

已知問題:

- 連續加減的時候會出錯
